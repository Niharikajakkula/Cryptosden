# Cryptosden Product Overview

Cryptosden is a comprehensive, full-stack cryptocurrency platform combining trading, community, portfolio management, and admin tools.

## Core Purpose
A complete crypto ecosystem providing users with market data, trading capabilities, community engagement, and administrative oversight in a single platform.

## Key Modules

**Authentication & User Management**
- Multi-role system (User → Trader → Admin)
- Two-factor authentication (TOTP with backup codes)
- KYC verification with document upload
- OAuth integration (Google & Facebook)
- Account security with lockout protection

**Wallets & Transactions**
- Multi-currency wallets (Bitcoin, Ethereum, altcoins)
- Secure transaction processing (deposits, withdrawals, transfers)
- Real-time balance tracking
- Complete transaction history

**Trading Platform**
- Advanced order types (Market, Limit, Stop-Loss)
- Real-time order book
- Portfolio analytics and P&L tracking
- Risk assessment and scoring

**Community & Social**
- Discussion forums with voting
- Interactive polls with analytics
- Reputation system
- Content moderation tools

**Admin Dashboard**
- Platform analytics and monitoring
- User management and role administration
- Transaction oversight and fraud detection
- Content moderation and reporting

## Market Data Features
- Real-time cryptocurrency prices from CoinGecko API
- Trust Score algorithm (proprietary 0-100 rating)
- Emotional Volatility Index (EVI) for market sentiment
- Automatic updates every 60 seconds (1 minute)
- Historical charts with technical analysis

## AI Integration
- Rule-based AI chatbot for platform guidance
- Cryptocurrency assistance and trading help
- OpenAI integration ready (optional)
- Real-time market insights

## Design Philosophy
- Dark theme with cyan branding (#06b6d4)
- Glass-morphism effects for modern UI
- Mobile-first responsive design
- Professional trading interface
- Smooth animations and transitions
